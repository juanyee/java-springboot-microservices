# Spring Boot Microservices

## Description
This project contains the following modules:
- api-gateway
- configuration-server
- discovery-server
- inventory-service
- notification-service
- order-service
- product-service

## Prerequisites
- Maven 3
- Java 11

## Technologies
- API Gateway
- Config Server
- Eureka
- Vault
- Bootstrap
- Actuator
- KeyCloak
- Bus
- RabbitMQ
